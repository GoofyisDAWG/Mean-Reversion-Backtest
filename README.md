# Mean-Reversion-Backtest
Z-score mean reversion strategy backtested across 6 global assets. Grid searches 20 param combos (window 10–60 days, threshold 1.0–2.5) with Sharpe heatmap and walk-forward train/test split (2016–2020 / 2021–2026). Strategy 5 of 5 toward an automated multi-strategy asset screener.
